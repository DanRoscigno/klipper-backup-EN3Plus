## My notes

I saw errors when I tried to install using `curl -fsSL get.klipperbackup.xyz | bash`, so I opened [https://get.klipperbackup.xyz/](https://get.klipperbackup.xyz/) in a browser and copied the script from there and ran:

```sh
git clone https://github.com/Staubgeborener/klipper-backup.git ~/klipper-backup
~/klipper-backup/install.sh
```

The instructions for creating a [repository and fine-grained personal access token](https://klipperbackup.xyz/installation/#prepare-github-repository) are excellent.
